Privacy Policy

This application integrates Mattermost with Zoom to allow users to create and manage meetings within Mattermost.

No personal data is collected, stored, or shared outside of Zoom and Mattermost systems.

OAuth tokens are used only for authorized API communication with Zoom and are stored securely within the Mattermost server.

No third-party services are used to process or store user data.
